# 企业内部 LLM Agent 平台

项目是干嘛的（一句话）
统一构建、部署、治理企业内部基于 LLM 的 Agent 应用平台。

你为什么要做它（动机）
避免重复造轮子与不可控的智能体扩散，提供统一的模型接入、知识治理、安全合规与成本可控能力。

它不是干嘛的（边界）
不是面向外部客户的公有 SaaS；不是单一业务线的专属应用；不替代企业现有核心业务系统。

当前阶段（MVP / PoC / 玩具）
PoC（概念验证阶段）。

使用方式（哪怕是伪的）
1. 管理员创建模型接入与权限策略
2. 业务方配置 Agent（Prompt、工具、知识源）
3. 通过 Web/IM/SDK 调用 Agent 完成任务
